# RTOSPaint
Video Demo for RTOS Paint - code is not uploaded for academic integrity

Real-Time Operating Systems (RTOS) Paint
• Developed a real-time operating system on a TIVA-C microcontroller using C.
    -> the RTOS paint is running on the RTOS made by me. 
• Integrated SPI, I²C, and DAC communication to interface with an ST7789 display, buttons, and joystick.
• Created an interactive drawing application supporting multiple colors, shapes, and tools
